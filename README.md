# Hermes 学习路线

从入门到精通 Hermes Agent 的结构化学习路径 — 6 个阶段，从安装第一行命令到写插件、贡献代码。

## 快速开始

```bash
git clone git@github.com:dao-ai/hermes-learning.git
cd hermes-learning
python3 -m http.server 8765
# 浏览器打开 http://localhost:8765
```

或直接访问：**[dao-ai.github.io/hermes-learning](https://dao-ai.github.io/hermes-learning/)**

## 学习阶段

| 阶段 | 内容 | 时间 |
|------|------|------|
| 1 | 安装和第一次对话 | 1 小时 |
| 2 | 日常工具使用（文件、命令、搜索、Git） | 1-2 天 |
| 3 | 深入理解配置和模型 | 1 周 |
| 4 | 技能（Skills）和记忆系统（Memory） | 1-2 周 |
| 5 | 进阶能力（委托、后台、定时、多平台） | 3-4 周 |
| 6 | 开发者级别（写插件、写技能、贡献代码） | 持续 |

## 网站功能

- 进度追踪（localStorage 持久化）
- 可折叠阶段卡片
- 每个阶段有检查清单
- 代码块一键复制
- 暗色主题，响应式设计
- 打印友好

## 关于 Hermes

[Hermes Agent](https://github.com/NousResearch/hermes-agent) 是 Nous Research 开发的开源 AI 代理框架，支持终端、消息平台和 IDE。兼容 20+ LLM 提供商，拥有技能系统、持久记忆和多平台网关。
